Проект посвящён изучению закономерностей, определяющих успешность игр. 

Цель: используя исторические данные о продажах компьютерных игр, оценки пользователей и экспертов, данные о жанрах и платформах,
выявить закономерности, определяющие успешность игры. 

Выявлены параметры, определяющие успешность игры в разных регионах мира. Проведена предобработка данных, анализ. Выбран актуальный
период для анализа. Составлены портреты пользователей каждого региона. 

Выполнена проверка гипотез. При анализе использован критерий Стьюдента для независимых выборок.
